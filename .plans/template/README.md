# Template Plans

This directory contains plans related to the **agentic-vibe-coding template** itself —
agent definitions, dashboard features, workflow protocols, and other template infrastructure.

> These plans are **synced from upstream** when a child project runs `sync-template`.
> They are NOT protected by `merge=ours` in `.gitattributes`.

## What Belongs Here

- Plans for features of the multi-agent template (dashboard, agent protocols, sync scripts)
- Architectural decisions about the template's own infrastructure
- Workflow protocols (e.g., the strict delegation protocol)

## What Does NOT Belong Here

- Plans for project-specific features → use `.plans/project/` instead
- Application architecture, business logic, or project requirements
