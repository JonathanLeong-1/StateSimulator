# Project Agent Logs

This directory contains agent session logs from work done **on the project** built
using the agentic-vibe-coding template.

> These logs are **protected from upstream sync** via `merge=ours` in `.gitattributes`.
> Template updates will never overwrite project-specific logs.

## What Belongs Here

- Session logs from agents working on project features, bug fixes, and refactors
- Execution details from team leads implementing project architecture plans

## What Does NOT Belong Here

- Logs from template infrastructure work → those live in `.agent-logs/template/`
