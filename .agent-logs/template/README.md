# Template Agent Logs

This directory contains agent session logs from work done **on the template itself** —
dashboard features, agent protocol changes, workflow infrastructure, etc.

> These logs are **synced from upstream** when a child project runs `sync-template`.
> They provide context for how the template was built and evolved.

## What Belongs Here

- Logs from agents working on template infrastructure (dashboard, agent definitions, sync scripts)
- Session summaries from template-level architectural work

## What Does NOT Belong Here

- Logs from project-specific feature work → use `.agent-logs/project/` instead
